# 红薯自动

小红书关注、私信、评论自动回复的Xposed模块

# 支持以下功能

- 关注自动回复文本
- 私聊自动回复文本
- 私聊自动回复图片
- 评论自动回复文本
- 进群自动回复文本

# 适配版本

8.38.0、8.29.0、7.80.0
